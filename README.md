# Marine Navigator (Marine app)

Interaktyvi navigacijos programa jūrai / laisvai veikianti žiniatinklio aplikacija.

## Funkcijos
- Realaus laiko GPS vieta (naudojant `navigator.geolocation.watchPosition`)
- Maršruto taškų pridėjimas pagal paspaudimą žemėlapyje (waypoint mode)
- Maršruto atstumo skaičiavimas
- Maršruto išvalymas
- Offline zonos išsaugojimas / įkėlimas (localStorage)
- Žemėlapio sluoksniai (base layer + sonar/depth overlay)
- **Kalbos keitimas (LT/EN)**
- **Dieninis / Naktinis režimas (Day/Night theme)**

## Kaip paleisti
1. Paprasčiausias paleidimas: atsidarykite `index.html` naršyklėje.
2. Rekomenduojama (dėl geolocation ir kai kurių naršyklės saugumo apribojimų): 
   - paleiskite local serverį, pvz. VSCode Live Server.

### Greitas startas
- Atidarykite projektą VSCode ir naudokite *Live Server* ant `index.html`.
